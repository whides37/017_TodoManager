title(str)

## ReactPython と連携する
- React（フロント）
- FastAPI / Flask（バックエンド）
- fetch で通信

## React起動手順
① プロジェクト作成
ターミナルで好きなフォルダに移動して下記実行：
npm create vite@latest my-todo-app --template react

② プロジェクトに入る
cd my-todo-app

③ 必要なパッケージをインストール
npm install

④ 開発サーバーを起動
npm run dev
